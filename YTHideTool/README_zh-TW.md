<h1> youtube 隱藏工具 </h1>

</div>

---

> **大致功能說明**

<h3>1. 初始載入觸發 : </h3>

* 有些影片結尾會出現推薦播放擋住視線 , 該腳本會自動隱藏該元素 , 當滑鼠移至影片區塊內時 恢復

<h3>2. 快捷觸發 : </h3>

* 透過快捷鍵去隱藏播放介面的一些元素 , 並記住設置在下一次點入影片時自動隱藏

<h3>3. 使用上可能的問題 : </h3>

* 當不完全隱藏, 或快捷隱藏失效, 這是因為 youtube 的加載方式, 導致腳本沒完整重新加載, 因此需重新整理頁面

---

> **功能說明**

```
在首頁無法使用這些功能 , 只有影片觀看頁面可使用 
(提示文字自適應: 繁中, 簡中, 日語, 英語, 韓語)

(Alt + 1) : 隱藏右側影片推薦

(Alt + 2) : 隱藏留言區

(Alt + 3) : 隱藏功能選項

(Alt + 4) : 隱藏播放清單資訊 (只在播放清單資訊界面可用)

(Alt + T) : 隱藏標題文字

(Ctrl + Z) : 使用極簡化 (隱藏上述除了 播放清單以外 全部)
```

> **開發說明**

```
只要將推薦隱藏起來, 就不會一直被演算法推薦, 導致一直看都停不下來
```
